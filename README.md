## Install 
1. Clone the repository
2. Install the database (database.sql)
3. Fill the user/pass fields for database in src/settings.php
4. Update dependencies
```sh
composer update
```
5. Start app 
```sh
composer start
```
6. Sign in with user@pass/pass
